# 🔍 Port-Scanner (Basic)

Its a tool to scan ports on a computer network. This tool helps to learn about network security and how to find weaknesses in computer systems.

This project is something I did to learn more about cybersecurity.Actually, I wanted to understand how computers talk to each other and how to find security problems.

## About This Project

This Port Scanner checks if certain doors on a computer are open. These doors are like entry points on a computer and they can show what services are running on the system. Its very important for people who test computer security.

The tool looks at a range of these doors. Tells which ports are open.

## What It Can Do

- It can scan any computer address or website

- It finds doors in real time

- It is small and works quickly

- It is easy to understand, even for beginners

- I built it using Python, a programming language

## How It Works

1. Tell the tool what computer to look at

2. The tool checks all the doors from 1 to 1024

3. It tries to connect to each door

4. If a door is open it shows me away

## What I Used To Build It

- Python 3 a programming language

- A library called socket comes with Python. It helps with network communication.
